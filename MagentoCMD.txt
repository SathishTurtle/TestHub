
Config Git
----------

To Clone
-----------

git clone https://github.com/SathishTurtle/TestHub.git

To Add
----------

git add FILENAME

To Commit
-----------

git commit -m "comments" filename

To Push OR Pull
----------------

git push -u origin (branch name)

git pull -u origin (branch name)

To find git Status, git Branch
-------------------

git status

git Branch
